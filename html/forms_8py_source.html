86def check_password_complexity(form, field):
87 """
88 Verifica a complexidade da senha fornecida.
89
90 Esta função é utilizada como um validador para garantir que a senha atenda a critérios de complexidade
91 específicos. A função verifica se a senha contém pelo menos uma letra maiúscula e pelo menos um caractere
92 especial.
93
94 Parâmetros:
95 form (FlaskForm): O formulário que contém o campo da senha.
96 field (Field): O campo da senha que está sendo validado.
97
98 Levanta:
99 ValidationError: Se a senha não contiver pelo menos uma letra maiúscula ou um caractere especial.
100
101 Exemplo de uso:
102 # Esta função pode ser usada como um validador em um campo de senha em um formulário Flask.
103 password_hash = PasswordField('Senha', validators=[DataRequired(), check_password_complexity])
104 """
105 password = field.data
106 if not re.search(r'[A-Z]', password):
107 raise ValidationError('A senha deve conter pelo menos uma letra maiúscula.')
108 if not re.search(r'[\W_]', password):
109 raise ValidationError('A senha deve conter pelo menos um caractere especial.')

130
131 Validações:
132 - O campo 'email' exige um e-mail válido.

134 - Todos os campos obrigatórios devem ser preenchidos antes do envio do formulário.
135

137 Este formulário é utilizado em páginas relacionadas à autenticação, como login, cadastro e recuperação de senha.

155class RegisterForm(FlaskForm):
156 """
157 Formulário de registro de novos usuários.
158
159 Este formulário é usado para capturar e validar os dados necessários para o registro de um novo usuário no sistema.
160
161 Campos:
162 - username: Campo de texto para o nome completo do usuário. É obrigatório.
163 - Validações:
164 - `DataRequired`: Certifica-se de que o campo não está vazio.
165 - email: Campo de texto para o email do usuário. É obrigatório e deve ser um endereço de email válido.
166 - Validações:
167 - `DataRequired`: Certifica-se de que o campo não está vazio.
168 - `Email`: Garante que o valor inserido é um email válido.
169 - password: Campo para a senha do usuário. É obrigatório e deve ter pelo menos 6 caracteres.
170 - Validações:
171 - `DataRequired`: Certifica-se de que o campo não está vazio.
172 - `Length`: Garante que a senha tenha no mínimo 6 caracteres.
173 - confirm_password: Campo para confirmar a senha do usuário. É obrigatório e deve corresponder ao valor do campo `password`.
174 - Validações:
175 - `DataRequired`: Certifica-se de que o campo não está vazio.
176 - `EqualTo`: Garante que o valor corresponde ao campo `password`.
177 - submit: Botão para enviar o formulário.
178
179 Uso:
180 Este formulário é renderizado em templates para capturar as informações do usuário e validá-las antes de criar um novo registro no banco de dados.
181
182 Exemplo:
183 >>> form = RegisterForm()
184 >>> form.validate_on_submit()
185 """
186
187 username = StringField(
188 'Nome Completo',
189 validators=[
190 DataRequired(message="O campo 'Nome Completo' é obrigatório.")
191 ],
192 )
